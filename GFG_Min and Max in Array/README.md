Given an array arr[]. Your task is to find the minimum and maximum elements in the array.

Examples:

Input: arr[] = [1, 4, 3, -5, -4, 8, 6]
Output: [-5, 8]

Input: arr[] = [12, 3, 15, 7, 9]

Explanation: minimum and maximum element of array are 3 and 15.
